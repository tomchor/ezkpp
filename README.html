This is an unofficial guide is aimed at providing additional information not covered by the official KPP manual. We focus on the latest version of the software, release 2.2.3, which can be freely downloaded at its official webpage.

The source for the documentation can be found at its github page and the online html version can be accessed here.

We recommend any person reading this guide to keep a copy of the original manual (which can be downloaded here) since this guide is not meant to replace the original manual, but to supplement it.
